Remove Text Punctuation :

Remove Text Punctuation means Remove all special characters from line or word like, [ !, @, #, $, %, ^, &, *, (, ),? ].After Removing Punctuation frequency of word is decreased.

Example :

Input : hello #%$^

Output : hello

UpperCase :

It Will Covert All Characters of Text in UpperCase (Capital)

Example :

Input : hello there !

Output : HELLO THERE !

New Line Remover :

It Will Remove All New line in Text and return in one line.

Example :

Input : hello there ! how are you ? welcome to my website !

Output : hello there ! how are you ? welcome to my website !

Remove Extra Space :

It Will Remove all Extra space from Text and return organized text.

Example :

Input : hello hii welcome

Output : hello hii welcome

Character Counter :

It Will return total No. of characters from text.

Example :

Input : hello there !

Output : 13

Word Counter :

It Will return total No. of Words from text.

Example :

Input : hello there welcome!

Output : 3

Capitalized Text :

It Will return converted capitalized all first character of text.

Example :

Input : hello there !

Output : Hello There!
